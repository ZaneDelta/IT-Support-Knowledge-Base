# Outlook Login Issue Troubleshooting

## Common Symptoms
- Unable to sign in to Outlook
- Repeated password prompts
- “Something went wrong” message
- Outlook stuck on loading profile
- Account disconnected from server

---

## Possible Causes
- Incorrect password
- Internet connectivity issue
- Corrupted Outlook profile
- Expired Microsoft credentials
- Office activation problem

---

## Troubleshooting Steps

### 1. Check Internet Connection
Make sure the device is connected to the internet properly.

Test connection:
```cmd
ping outlook.office.com
