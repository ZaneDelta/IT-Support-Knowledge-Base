# WiFi Troubleshooting Guide

## Common Symptoms
- Connected but no internet
- Slow connection
- Frequent disconnections

## Troubleshooting Steps

### 1. Restart Router
Turn off router for 30 seconds.

### 2. Forget and Reconnect Network
Settings → WiFi → Forget Network

### 3. Reset IP Configuration

```cmd
ipconfig /release
ipconfig /renew
ipconfig /flushdns
